# rock-paper-scissors-py

## About this Project

A simple command-line rock, paper, scissors game I worked on as a exercise.

I chose an Object-Oriented approach to keep the game logic separate from the user interface, making the code more scalable and easier to maintain.

## Installation

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/axhis8/rock-paper-scissors-py.git
    cd rock-paper-scissors-py
    ```

2.  **Run the application:**

    ```bash
    python main.py
    ```

## Features

- Play against a computer
- Choose how many Rounds you want to play
- Input validation with robust Error Handling
- Clean UI
- Ability to Play again
- Object Oriented Syntax
